Mac安装aria2

发表于 2020-11-14 更新于 2023-04-16 分类于 tools
本文字数： 1.5k 阅读时长 ≈ 1 分钟
Aria2 本质是一个下载工具，支持多协议、多来源，可以下载磁力链接和种子资源，几乎支持所有下载方式的工具

1. 博客推荐

雨月空间站

2. 下载aria2安装包

https://github.com/aria2/aria2/releases

Mac能直接安装的版本是 1.35.0

下载后直接安装

运行终端 aria2c，出现 aria2c: command not found

  1. 查找aria2c执行文件

  2. 软连接

3. 配置文件

  1. 下载配置文件，解压后放在 /Users/用户名中

  2. 文件夹命名为 .aria2，被称为隐藏文件，Mac中需要 Shift+Command+.来显示隐藏文件

  3. 终端中打开文件夹中的Aria2

    按键 i，进入编辑状态

    修改其中的 你的名字 为 你的用户名

    按键 esc，结束编辑

    输入 :wq，退出文件

  4. 打开 aria2.conf，将所有的 你的名字 换成 你的用户名

    可以使用 sublime Text，VSCode等软件，实现一键替换所有

  5. 双击 Aria2 执行文件

    如果出现listening，successfully等字样，即为成功

  6. 若想关闭连接，按键 control+c

4. 可视化页面

打开文件夹里 AriaNG文件夹，里面有个 index.html，用浏览器打开

注意：要保持上一步 aria2的开启

5. 修改配置

确定下载路径，同时下载数等

  • 可以在AriaNg界面在修改
  • 也可以在aria.conf中修改

6. 自动操作APP

如果希望一次性打开 网页+Aria服务，可以使用Mac中的自动操作

  1. control+空格键 打开 聚焦搜索，搜索 自动操作.app

  2. 选择 新建文稿 -> 工作流程

  3. 将 aria执行文件 和 AiraNg的index.html 放入 操作台

  4. 指定操作 打开访达项目

  5. 将项目存储为app，放在桌面

  6. 查看这个自动流程app，右键显示包内容，找到它的图标，替换成 .aria2文件夹中的图标

  7. 下次双击这个app就可以自动打开网页和aria2服务

7. 网盘直链下载

7.1. 倍速播放

GlobalSpeed 视频速度控制

下载后解压，打开chrome浏览器的扩展程序 chrome://extensions/，打开 开发者模式，将解压的 crx 文件拖入扩展程序

实现倍速播放

7.2. 下载助手

下载 Tampermonkey Beta 油猴脚本

同样安装到chrome浏览器中

打开 Greasy 安装 百度网盘简易下载助手（直链下载复活版）

此时打开网盘，在网页中出现 简易下载助手按钮，勾选任一个文件，获取下载链接

双击创建的aria2.app，准备好环境后，点击 发送至Aria2，在index.html网页中就出现了下载进度
