10. 二分

  • [left,right] 内查找一个数

    while(left<=right) 返回A[mid]

    当left>right时，说明不存在，返回-1

  • [left,right) 内查找>=x或<=x的数

    while(left<right)

    当left=right时，说明找到数应该所在的位置 return A[left]

  • (left,right] 内查找一个数

    while(left+1<right)

    return A[right]

  • 计算函数值

    while(right-left>eps){ if(f(mid)>0){right=mid;} else{left= mid;} return mid;}
